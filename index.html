NodeBox

Simple NAS app & local website host for your home or office network.

About NodeBox

NodeBox is a powerful and lightweight local content management and serving tool. It enables easy sharing, streaming, and organizing of files and media on your home or office network — without requiring internet access or complex setup.

Features

  • Serve and manage multiple local sites: Organize files and media into mini-websites or shared folders with an easy dashboard.
  • Built-in media player: Stream audio and video directly from your browser with seamless playback over your local network.
  • Storage quota management: Control disk usage by setting max storage sizes per shared site folder.
  • User authentication and security: Protect your sites and dashboard with secure login credentials.
  • Automatic local network discovery (mDNS/Bonjour): Connect by name (e.g., yoursite.local) without needing IP addresses.
  • Cross-platform and lightweight: Runs on Linux, Windows, and macOS with minimal resource usage. Packaged as standalone executables.
  • Extensible and customizable: Advanced users can add scripts and tweak configurations.
  • Offline-first, zero internet dependency: Fully functional without internet — perfect for privacy and offline environments.

Usage

Access the NodeBox dashboard in your browser at:

  • http://127.0.0.1:5000 (on the device running NodeBox)
  • http://<your-device-ip>:5000 (from any other device on the same local network)

Tip: If you're running NodeBox on a Raspberry Pi, visit your Pi's IP address at port 5000 (e.g., http://192.168.1.100:5000).

Use the dashboard to create and deploy sites, manage media, and configure settings like storage quotas.

Installation & Startup

  1. Clone the repo using git: 
    git clone https://github.com/imcaca12/NodeBox.git
  2. Open the NodeBox folder.
  3. Make the NodeBox file executable: 
    chmod +x NodeBox
  4. Run the startup script to enable NodeBox to launch on boot: 
    ./startservice.sh
  5. (Optional) Reboot your device and visit http://<your-Pi-IP>:5000 in your browser.
  6. If you don’t reboot, start NodeBox manually with: 
    ./NodeBox
